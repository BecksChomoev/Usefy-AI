# Usefy-AI

A Spring Boot application built as part of the GH Academy Internship.

## Tech Stack
- Java 17+
- Spring Boot
- Spring Data JPA
- H2 Database (in-memory)
- Maven
- Lombok

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `HelloSpringApplication.java`
4. Visit `http://localhost:8080/hello`

## H2 Console
Available at `http://localhost:8080/h2-console`
- JDBC URL: `jdbc:h2:mem:testdb`
- Username: `sa`
- Password: admin
